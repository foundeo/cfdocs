# sin

 Calculates the sine of an angle that is entered in radians.

```javascript
sin(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Sine of 0.3

```javascript
sin(0.3)
```

### Expected Result: 0.295520206661
