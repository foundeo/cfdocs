# year

 From a date/time object, gets the year value.

```javascript
year(date)
```

```javascript
returns numeric
```

## Member Function Syntax

```javascript
date.year()
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| date | date | Yes |  |  |

## Year of a datetime object

```javascript
dt = createdatetime(2016,1,1,5,30,25);
y = year( dt );
writeoutput( y );
```

### Expected Result: 2016
