# asin

Determines the arcsine of a number. The arcsine is the angle whose sine is `number`.

```javascript
asin(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Arcsine of 0.3

```javascript
asin(0.3)
```

### Expected Result: 0.304692654015
