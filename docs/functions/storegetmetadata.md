# storeGetMetadata

 Returns the metadata related to the object or bucket.

```javascript
storeGetMetadata(url);
```

```javascript
returns struct
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| url | string | Yes |  | Amazon S3 URLs (bucket or object). |
