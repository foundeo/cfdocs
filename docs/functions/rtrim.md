# rTrim

 Removes spaces from the end of a string.

```javascript
rTrim(String)
```

```javascript
returns string
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| String | string | Yes |  | A string or a variable that contains one |

## Right Trim

```javascript
'>' & rTrim('    CFDocs    ') & '<'
```

### Expected Result: >    CFDocs<
