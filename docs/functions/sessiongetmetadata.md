# sessionGetMetadata

 Returns meta data related to the current session.

```javascript
sessionGetMetadata();
```

```javascript
returns struct
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| region | string | Yes |  | Name of the cache region. |

## Tag Syntax

```javascript
<cfset metaData = sessionGetMetadata() >
```
