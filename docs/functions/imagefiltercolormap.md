# imageFilterColorMap

Used to provide a colormap as argument of the imageFilter function

```javascript
imageFilterColorMap(type [, lineColor1] [, lineColor2])
```

```javascript
returns any
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| type | string | Yes |  |  |
| lineColor1 | string | No |  |  |
| lineColor2 | string | No |  |  |
