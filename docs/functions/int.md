# int

 Calculates the closest integer that is smaller than number.

```javascript
int(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Closest integer less than 1.8

```javascript
int(1.8)
```

### Expected Result: 1
