# sslCertificateInstall

Installs certificates of a specific host into the JVM certificate store

```javascript
sslCertificateInstall(host [, port])
```

```javascript
returns void
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| host | string | Yes |  |  |
| port | numeric | No | 443 |  |
