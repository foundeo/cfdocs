# quarter

 Calculates the quarter of the year in which a date falls.

```javascript
quarter(date)
```

```javascript
returns numeric
```

## Member Function Syntax

```javascript
date.quarter()
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| date | date | Yes |  |  |

## Quarter of a datetime object

```javascript
dt = createdatetime(2016,1,1,5,30,25);
q = quarter( dt );
writeoutput( q );
```

### Expected Result: 1
