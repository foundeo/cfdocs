# isIPV6

Checks if given IP is of IPV6 format

```javascript
isIPV6(value)
```

```javascript
returns boolean
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| value | string | Yes |  | IP address to check |

## isIPV6 Example

```javascript
isIPV6("127.0.0.1");
```

### Expected Result: YES
