# lTrim

 Removes leading spaces from a string.

```javascript
lTrim(String)
```

```javascript
returns string
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| String | string | Yes |  |  |

## Left Trim

```javascript
'>' & lTrim('    CFDocs    ') & '<'
```

### Expected Result: >CFDocs    <
