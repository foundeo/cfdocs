# lCase

 Converts the alphabetic characters in a string to lowercase.

```javascript
lCase(String)
```

```javascript
returns string
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| String | string | Yes |  |  |

## lowercase CFDocs Rock

```javascript
lCase('CFDocs Rock')
```

### Expected Result: cfdocs rock
