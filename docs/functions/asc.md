# asc

Determines the ASCII value of a character.

```javascript
asc(string)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| string | string | Yes |  |  |

## character and value of Korean currency symbol ( unicode number 8361 )

```javascript
x = chr( 8361 ); 
 writeoutput( 'character: #x#' ); 
 x = asc( x ); 
 writeoutput( '<br>Unicode value: #x#');
```
