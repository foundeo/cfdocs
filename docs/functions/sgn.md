# sgn

 Determines the sign of a number.

```javascript
sgn(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Sign of any number greater than 0

```javascript
sgn(0.3)
```

### Expected Result: 1

## Sign of 0

```javascript
sgn(0)
```

### Expected Result: 0

## Sign of any number less than 0

```javascript
sgn(-0.3)
```

### Expected Result: -1
