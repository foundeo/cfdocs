# round

 Rounds a number to the closest integer.

```javascript
round(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Round 1.56

```javascript
round(1.56)
```

### Expected Result: 2

## Round 1.49

```javascript
round(1.49)
```

### Expected Result: 1

## Round -0.9

```javascript
round(-0.9)
```

### Expected Result: -1
