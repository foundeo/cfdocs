# structIsOrdered

Check if a struct is ordered

```javascript
structIsOrdered()
```

```javascript
returns boolean
```

## Member Function Syntax

```javascript
struct.isOrdered()
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| struct | struct | Yes | false | Struct that should be tested whether it is ordered |

## Check if a struct is ordered

```javascript
structNew("ordered").isOrdered()
```

### Expected Result: YES
