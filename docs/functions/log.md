# log

 Calculates the natural logarithm of a number. Natural
 logarithms are based on the constant e (2.71828182845904).

```javascript
log(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |
