# trim

Removes leading and trailing spaces from a string.

```javascript
trim(String)
```

```javascript
returns string
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| String | string | Yes |  | A string or a variable that contains one |

## Trim

```javascript
'>' & trim('    CFDocs    ') & '<'
```

### Expected Result: >CFDocs<
