# abs

 Absolute-value function. The absolute value of a number is
 the number without its sign.

```javascript
abs(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Absolute Value of -4.3

```javascript
abs(-4.3)
```

### Expected Result: 4.3
