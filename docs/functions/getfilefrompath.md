# getFileFromPath

 Extracts a filename from an absolute path.

```javascript
getFileFromPath(path)
```

```javascript
returns string
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| path | string | Yes |  |  |

## Basic Example

```javascript
getFileFromPath("C:\temp\file.txt")
```

### Expected Result: file.txt
