# atn

Arctangent function. The arctangent is the angle whose tangent is `number`.

```javascript
atn(number)
```

```javascript
returns numeric
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| number | numeric | Yes |  |  |

## Arctangent of 0.3

```javascript
atn(0.3)
```

### Expected Result: 0.291456794478
