# isEmpty

Returns whether or not a given value is "empty". 

```javascript
isEmpty(value)
```

```javascript
returns boolean
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| value | any | Yes |  |  |

## isEmpty Example

Returns a boolean based on the given value. Strings, Arrays, Structs and queries are accepted data types.

```javascript
isEmpty( {} )
```

### Expected Result: YES
