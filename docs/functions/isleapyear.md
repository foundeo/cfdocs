# isLeapYear

 Determines whether a year is a leap year.

```javascript
isLeapYear(year)
```

```javascript
returns boolean
```

## Argument Reference

| Name | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| year | numeric | Yes |  |  |

## Is the current date in a leap year?

```javascript
date = now();
isLeapYear = isLeapYear(year(date));
writeOutput(isLeapYear);
```
