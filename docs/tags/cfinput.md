# cfinput

Used within the cfform tag, to place radio buttons, check boxes, or text boxes on a form. Provides input validation for the specified control type.

### Syntax

```html
<cfinput name="">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfinput(name="");
```
